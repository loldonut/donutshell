# DonutShell

A Custom Shell I made in Python for fun, which means it sucks lmao

_This package is not meant for daily use._

## Installation

- Using `PIP`

```
pip install donutshell
```

- By cloning the repo

```sh
git clone https://github.com/loldonut/donutshell
cd donutshell/donutshell

python .
# or
py .
```
